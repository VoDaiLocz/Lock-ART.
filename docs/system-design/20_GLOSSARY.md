# 20. Glossary

- **Protection Score**: chỉ số nội bộ phản ánh mức drift chống bắt chước.
- **Profile**: preset tham số cho pipeline bảo vệ.
- **Collective mode**: bảo vệ theo nhóm subject-set.
- **Fidelity**: mức giữ chất lượng gần ảnh gốc.
- **Robustness transform**: biến đổi để đánh giá độ bền hiệu ứng bảo vệ.
- **Benchmark manifest**: mô tả đầu vào/cấu hình cho chạy benchmark.
