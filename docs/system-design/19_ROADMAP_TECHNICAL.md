# 19. Technical Roadmap (Design-level)

- Chuẩn hóa report schema v1.
- Tối ưu runtime CPU cho batch vừa.
- Bổ sung validation path/input chặt hơn.

- Plug-in attack API ổn định.
- Benchmark dashboard nhẹ từ JSON artifacts.
- Regression suite theo profile.

## Long-term

- Khả năng chạy distributed evaluation.
- Cross-model robustness evaluation chuẩn hóa.
