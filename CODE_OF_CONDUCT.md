# Code of Conduct

## Our Commitment

We are committed to making participation in the AuraLock community a respectful,
constructive, and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and professional in discussions.
- Assume good intent, and ask clarifying questions before escalating.
- Give actionable, evidence-based technical feedback.
- Focus critique on ideas and implementation, not individuals.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks, trolling, or hostile communication.
- Publishing private information without permission.
- Deliberate misinformation about benchmark outcomes.

## Scope

This Code of Conduct applies to:

- repository issues,
- pull requests and reviews,
- discussion threads,
- and related project communication spaces.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that
violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact maintainers through private channels.
For security-related concerns, use the process in [SECURITY.md](SECURITY.md).
